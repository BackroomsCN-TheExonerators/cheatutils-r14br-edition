World Download

Downloads chunks from multiplayer server into your singleplayer world.
To start downloading:
- Create new singleplayer world (or use existing one)
- Enter directory name of singleplayer world save directory (you can find it in "/saves" directory)
- Press "Start" button (no need to be logged in multiplayer server)
- Start loading server chunks by moving or logging in.
- Use Exploration Mini Map to see which chunks were saved to disk (dotted square)
- Press "Stop" button (don't forget to Stop before loading singleplayer world!)
Module only saves original chunks sent by server. For example, if you loaded chunk, built something here - this will not be saved. You need to unload/load this chunk to save new changes.

Note: some servers use custom dimensions, for example for spawn region. In this case region files will be saved into "/dimensions/somename1/somename2/region". You can manually copy region files into root "/region" directory to see them in your single player world.
