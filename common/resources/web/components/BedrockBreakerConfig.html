Bedrock Breaker (BETA)

Breaks bedrock with single button click.
Conditions:
  • Netherite Pickaxe Efficiency 5 on hotbar
  • Some amount of pistons on hotbar
  • Some amount of redstone torches on hotbar
  • Haste 2 effect from beacon (you should be able to instamine pistons)
How to use:
  • Add bedrockBreaker.process(); to keybinding
  • Target bedrock block
  • Ensure there are 2 empty blocks above bedrock
  • Ensure there is 1 block on the side of bedrock. Redstone torch will be placed on top of this block.
  • Press hotkey
  • If something goes wrong you should see system message
